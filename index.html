// 06 — Build Log

Projects

P-01 / Ongoing

AI-Based Automotive Gearbox Fault Identification

Implemented a machine learning-based diagnostic system to identify gearbox faults from vibration data. Signal processing, feature extraction, and fault classification in MATLAB for predictive maintenance.

P-02

IoT-Based Fire Alarm System

IoT-enabled fire detection system using ESP32, IR flame sensor, MQ-2, and DHT11. Integrated Blynk app alerts, LED indicators, and buzzer for real-time hazard response.

P-03

Thermal Engineering — Biofuel Blend Study

Experimental study on biofuel blends in IC engines focusing on emissions, combustion behavior, and thermal efficiency. Tested ethanol-petrol blends on a Maruti rig.

P-04

SAE ISS Autonomous Drone

Led a 10-member team designing an autonomous drone with multi-sensor fusion (GPS, LiDAR, ultrasonic), control algorithms, and mission-based scripting using Pixhawk and Raspberry Pi.

P-05

Sustainable Aviation Fuel (SAF) Research

Developed a syringe pump to simulate Fischer-Tropsch bubble reactions for SAF production. Collaborated with CSIR Dhanbad and Microfluidics Lab at CET Trivandrum. District finalist at Dreamvestor.

P-06

Mechanical Hand (Scrap-Based)

Engineered a functional mechanical hand using recycled components, replicating human joint motion through mechanical linkages and actuation design. First place at tech fest.

P-07

IoT in Agriculture

Sensor-integrated system to monitor real-time soil moisture, enabling precision irrigation and resource optimization. Built at IRTC Mundur with automated sunlight and irrigation control.

P-08

ROS Package Development (Internship)

Built and tested ROS nodes for real-time robotic control. Developed packages for sensor data handling, actuator control, and autonomous navigation in Python.

P-09

Python–MySQL Social App Prototype

Demo application with integrated front-end/back-end architecture. Showcasing cross-disciplinary software development with user auth and dynamic content management.
